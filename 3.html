LA GENÈSE : GESTION DE CRISE

LA PROBLÉMATIQUE

  • Contexte tendu : Gestion de forcenés ou crises majeures par les PSIG.
  • Poste de Commandement (PCT) : Nécessité absolue pour la sécurité et la coordination.
  • Méthode actuelle : Fil d'ariane manuscrit sur feuille blanche ou canevas papier.
  • Risques : Ratures, illisibilité, perte d'information, difficulté de transmission.

LA RÉPONSE NUMÉRIQUE

  • Usage simple : Compatible Ubiquity (PC) ou Smartphone à défaut.
  • Clarté : Main courante propre, modifiable sans rature.
  • Standardisation : Code couleur tactique automatique (Adversaire, Inter, AO...).
  • Diffusion : Export PDF instantané pour la VH, le CORG ou les unités IS.

POINTS FORTS TECHNIQUES

LOCAL STORAGE

Sécurité maximale : aucune donnée ne sort du terminal. Tout est stocké localement dans le navigateur. Pas de serveur, pas de cloud.

TRANSFERT QR

Innovation majeure : Transfert de la main courante d'un appareil à l'autre via QR Code animé, sans aucune connexion réseau (Mode Avion).

MODE TP

Adaptabilité : Bascule entre un mode Forcené (Adversaire/Inter) et un mode "Tueries Planifiés" pour identifier des intervenants spécifiques (Négociateur, Antenne GIGN...).
